 ## This is just teaching Vector and Matrix Transpose
- For Numpy the Code is
  ```
  import numpy as np

  nv=np.array([[1,2,3]])
  print(nv),print(" ")

  print(nv.T), print(" ")

  nvT=nv.T
  print(nv.T)

  nv=np.array([[1,2,3],[4,5,6])
  print(nv),print(" ")

  print(nv.T), print(" ")

  nvT=nv.T
  print(nv.T)
  ```
